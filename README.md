# 亡途荒夜 (Yongye)

极难灾变生存玩法 · Minecraft **Fabric 1.21.1**

> 白天跑图，夜晚逃命，永夜追杀；靠**装备血量 + 技能书(V65535)+ 背包神器 + 随机掉落**反向变强。

本仓库当前进度:**Phase 0 ~ Phase 4 全部代码已落地**(永夜、追杀、随机任务、背包神器、高血量反制、指令系统)。

## 一、当前已实现(Phase 0 ~ Phase 4)

### Phase 0 — 工程骨架
- Fabric Loom 工程(Gradle 8.10.2 wrapper 已内置)

## 二、构建

需要 **JDK 21**。

```bash
./gradlew build
```

产物在 `build/libs/yongye-0.1.0.jar`。把它(以及 Fabric API、Fabric Loader)丢进 `mods/` 即可。

> ⚠️ 说明:本工程在沙箱内编写,沙箱网络未放行 Fabric/Minecraft 的 Maven 源,因此**未能在沙箱实际跑通 `./gradlew build`**。代码按 1.21.1 + Fabric API 的正确 API 编写并逐项自检,但首次本地构建若遇到个别版本/映射不匹配,按 `gradle.properties` 顶部注释微调 `yarn_mappings` / `fabric_version` 即可(参考 https://fabricmc.net/develop )。

## 三、配置

首次运行生成 `config/yongye.json`,可调项包括:

- 总开关:`enableMobEnhancement` / `enableArmorHealthBonus` / `enableRandomLoot`
- 怪物增强:血量/攻击/移速倍率、击退抗性、感知范围、随机药水概率
- 套装加成:6 档材质的额外最大生命
- 技能书:封顶等级、三档材料门槛
- 随机掉落:5 档品质概率、生命碎片掉率、是否要求玩家击杀

## 四、Phase 3 / Phase 4(新增)

### Phase 3 — 永夜灾变 / 随机任务 / 追杀
- **永夜五级**(暗潮/猎杀/围城/灾变/灭世):等级持久化到存档 `yongye_nightfall.json`;level≥1 锁定黑夜;等级提升精英概率与怪物锁定半径。**赎夜**降低一级。
- **随机任务**(带 Boss 血条计时):猎杀精英 / 守住据点(存活)/ 限时逃离;成功发奖励,失败提升永夜等级。
- **追杀 AI**(纯 tick 驱动,零 mixin):永夜下**锁定**玩家、撞墙时按档位(普通/精英/Boss 硬度阈值)**挖墙**破障、目标在上方时**爬墙**垂直追击。

### Phase 4 — 背包神器 / 高血量反制 / 指令
- **背包神器(10 种)**:放进背包即生效,同类只取最高等级,1~6 级(残破/完整/远古/神话/灾变/终焉)。含生命神像、铁壁核心、骨箭护符、巫毒净瓶、逃亡者之靴、掘墓人罗盘、不灭余烬、永夜之眼、饕餮心脏、世界锚石。神器 + 阶段材料可在工作台**升级**(`yongye:artifact_upgrade`)。
- **高血量反制**:Boss/精英/永夜怪命中玩家附加「最大生命百分比 + 真实(无视护甲)伤害」,并概率**禁疗**与**最大生命压制**;骨箭护符可概率免疫箭矢。
- **指令**(需 OP / 权限 2):
  - `/yongye nightfall status|<0-5>` 查看/设置永夜等级
  - `/yongye redeem` 赎夜(降一级)
  - `/yongye quest hunt|survive|flee` 给自己派发任务
  - `/yongye book <1-65535>` 获取血量技能书
  - `/yongye artifact <类型id> <1-6>` 获取神器(类型 id 见物品组英文名)

> 说明:为保证 1.21.1 下稳定编译并避免 mixin 复杂度,「铁壁核心 / 巫毒净瓶 / 禁疗」等少数效果采用近似实现(抗性效果 / 概率清除 / 饥饿压制食物回复),已在代码注释标注,可按需替换为更精确的伤害管线实现。


## 五、Phase 4 之后的扩展

- **6 本新技能书**(攻击/护甲/恢复/闪避/反伤/抗性):可右键学习、同级合成、普通/精英/Boss 三档掉落;`/yongye skillbook <类型> <等级>`。
